# Netflix Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Movies vs TV Shows
- Top Countries
- Top Directors
- Release Year Analysis
- Rating Analysis
- Data Filtering

## Files

- netflix_titles.csv
- netflix_cleaned.csv

## How to Run

```bash
pip install -r requirements.txt

```
